# Refinx Documentation

This folder contains documentation for the Refinx project, a chat interface built with Tauri and React.

## Setup Guide

- **[Setup Instructions](setup.md)**: Step-by-step guide to set up the project, including dependencies, Tauri CLI, and development server.

## Project Structure

- `src/`: React frontend code.
- `src-tauri/`: Rust backend code.
- `tauri.conf.json`: Tauri configuration file.

## Next Steps

- Design the chat UI in the React frontend.
- Implement API calls to an LLM in the Rust backend.
- Use Tauri's IPC for communication between frontend and backend.
